## 📌 Overview

An **AI-powered music recommendation system** that detects user emotions in real-time using computer vision and suggests songs accordingly.

💡 This project combines **Machine Learning + Computer Vision + Web Integration** to deliver a smart and interactive music experience.

---

## ✨ Features

🚀 Real-time Emotion Detection  
🎶 Mood-based Song Recommendation  
🔍 Dynamic YouTube Song Search  
🎨 Clean & Interactive UI  
⚡ Fast and Efficient System  

---

## 🧠 How It Works

1️⃣ Capture user face using webcam 📷  
2️⃣ Detect emotion using trained ML model 🤖  
3️⃣ Classify emotion (Happy, Sad, Angry, fear ,neutral etc.) 😎  
4️⃣ Recommend songs based on detected emotion 🎵
